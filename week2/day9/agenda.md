# Day 9: Linear Regression Theory

## 🎯 Goal

Understand how Linear Regression works and when to use it.

---

## 📚 Study Plan

### 1. Understand Regression Problems

* What is Regression?
* Regression vs Classification
* Real-world use cases

---

### 2. Learn Linear Regression Basics

* Simple Linear Regression
* Multiple Linear Regression
* Best Fit Line

---

### 3. Understand the Equation

* y = mx + b
* Slope (m)
* Intercept (b)

Practice solving a few examples manually.

* Mean Squared Error (MSE)

* Linear relationship

* Minimal outliers
* Constant variance

### 8. Visual Understanding

* Best fit line
* Scatter plot with regression line
* Underfitting vs Overfitting (revision)
